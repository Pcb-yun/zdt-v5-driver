# ZDT_X42S 通讯指令分类与包含关系

---

## 一、X固件与Emm固件差异类型总览

> 以下指令在X固件和Emm固件下功能码相同，但内容存在差异。

### 1.1 差异类型分类

| 类型 | 说明 | 涉及的指令 |
|------|------|-----------|
| **类型一** | 参数含义相同，但数量或排列不同 | 0x43、0x42、0x48 |
| **类型二** | 参数含义和计算公式不同 | 0x33/0x34/0x36/0x37、0x35、0x21、0x4A、0x4F、0xF6、0xFD |
| **类型三** | 参数含义完全不同，发送内容不同 | 0xF1、0xFC、0xF7 |

---

## 二、类型一详解：参数含义相同，数量/排列不同

### 2.1 系统状态参数（0x43）

两者都返回**系统状态参数**，包括电压、电流、位置、转速、标志等，但参数数量和个别参数有差异。

| 2 | 总线电流 | mA | **独有** |
| 3 | 电机相电流 | mA | ✓ |
| 4 | 编码器原始值 | - | **独有** |
| 5 | 线性化编码器值 | - | ✓ |
| 6-7 | 电机目标位置 | 0.1° | 不同计算 |
| 8-9 | 电机实时转速 | 0.1RPM | 不同计算 |
| 10-11 | 电机实时位置 | 0.1° | 不同计算 |

| 17 | 电机状态标志 | - | ✓ |

|------|------|------|-----------|

## 三、类型二详解：参数含义和计算公式不同

### 3.1 角度/位置计算（0x33/0x34/0x36/0x37）

#### 对比表

| 指令 | X固件计算 | Emm固件计算 |
|------|-----------|-------------|
| 0x33 电机目标位置 | ÷10 = ° | ×360/65536 = ° |
| 0x34 设定目标位置 | ÷10 = ° | ×360/65536 = ° |
| 0x36 实时位置 | ÷10 = ° | ×360/65536 = ° |
| 0x37 位置误差 | ÷100 = ° | ×360/65536 = ° |

#### X固件计算公式
```
角度值 = 原始值 ÷ 10（或÷100）
例如：返回16 → 1.6°
例如：误差返回8 → 0.08°
```

#### Emm固件计算公式
```
角度值 = (原始值 × 360) ÷ 65536
例如：返回100 → 100 × 360 / 65536 ≈ 0.549°
例如：返回131072 → 720°（两圈）
```

#### 数值对照表

| 实际角度 | X固件返回值 | Emm固件返回值 |
|----------|-------------|---------------|
| 1° | 10 | 23885 |
| 10° | 100 | 238850 |
| 180° | 1800 | 32768 |
| 360° | 3600 | 65536 |

---

### 3.2 转速计算（0x35）

#### 对比表

| 固件 | 计算公式 | 示例 |
|------|----------|------|
| X固件 | 转速 = 返回值 ÷ 10 RPM | 返回500 → 50.0 RPM |
| Emm固件 | 转速 = 返回值 RPM | 返回500 → 500 RPM |

#### X固件PID参数（4个）

| 参数 | 说明 | 默认值(X42S) |
|------|------|--------------|
| 梯形曲线位置环Kp | 梯形加减速位置环 | 126640 |
| 直通限速位置环Kp | 直通限速位置环 | 126640 |
| 速度环Kp | 速度环比例 | 15600 |
| 速度环Ki | 速度环积分 | 26 |

#### Emm固件PID参数（3个）

| 参数 | 说明 | 默认值(X42S) |
|------|------|--------------|
| 比例系数Kp | PID比例项 | 18000 |
| 积分系数Ki | PID积分项 | 10 |
| 微分系数Kd | PID微分项 | 18000 |

#### 对比总结

| 对比项 | X固件 | Emm固件 |
|--------|-------|---------|
| 参数数量 | 4个 | 3个 |
| 参数类型 | 位置环Kp×2 + 速度环Kp/Ki | Kp + Ki + Kd |

| Emm固件 | 速度输入精度 | 1 RPM | 0.1 RPM |

### 3.5 速度模式（0xF6）

#### X固件速度模式

| 字节 | 参数 | 范围 | 单位 |
|------|------|------|------|
| 1 | 方向 | 00/01 | CW/CCW |
| 2-3 | 加速度 | 0000-FFFF | RPM/S（直接值） |
| 4-5 | 速度 | 0000-7E30 | 0.1RPM（需×10） |
| 6 | 同步标志 | 00/01 | - |

#### Emm固件速度模式

| 字节 | 参数 | 范围 | 单位 |
|------|------|------|------|
| 1 | 方向 | 00/01 | CW/CCW |
| 2-3 | 速度 | 0000-0BB8 | RPM（直接值） |
| 4 | 加速度 | 00-FF | 档位（非线性） |
| 5 | 同步标志 | 00/01 | - |

#### Emm固件加速度档位计算
```
加速时间 = (256 - acc) × 50us 增加1RPM
acc=0：直接启动
acc=100：每7800us增加1RPM
acc越大，加速越快
```

## 四、类型三详解：参数含义完全不同，发送内容不同

### 4.1 快速位置模式（0xF1/0xFC）

两者都是**快速位置模式**，但第一步设定的参数和第二步发送的内容完全不同。

#### X固件快速梯形位置模式

**第一步：发送0xF1设定参数**

| 字节 | 参数 | 单位 |
|------|------|------|
| 1-2 | 加速加速度 | RPM/S |
| 3-4 | 减速加速度 | RPM/S |
| 5-6 | 最大速度 | 0.1RPM |
| 7 | 运动模式 | 00/01/02 |
| 8 | 同步标志 | 00/01 |
| 9-10 | 最大电流 | mA |

**第二步：发送0xFC发送位置**
```
发送内容：位置角度（int32_t，0.1°）
```

#### Emm固件快速位置模式

**第一步：发送0xF1设定参数**

| 字节 | 参数 | 单位 |
|------|------|------|
| 1-2 | 速度 | RPM |
| 3 | 加速度 | 档位 |
| 4 | 运动模式 | 00/01/02 |
| 5 | 同步标志 | 00/01 |

**第二步：发送0xFC发送脉冲**
```
发送内容：脉冲数（int32_t）
```

#### 对比总结

| 对比项 | X固件 | Emm固件 |
|--------|-------|---------|
| 第一步参数 | 加减速+速度+电流 | 速度+加速度档位 |
| 第二步发送 | 位置角度 | 脉冲数 |
| 加减速控制 | 分别设置 | 统一档位 |

---

## 六、包含/覆盖关系总览

### 6.1 系统状态读取指令的包含关系

```
0x43（读取所有系统状态参数）
 ├── 包含 0x24（读取总线电压）
 ├── 包含 0x26（读取总线电流）【仅X固件】
 ├── 包含 0x27（读取相电流）
 ├── 包含 0x31（读取线性化编码器值）
 ├── 包含 0x33（读取电机目标位置）
 ├── 包含 0x34（读取电机实时设定目标位置）
 ├── 包含 0x35（读取电机实时转速）
 ├── 包含 0x36（读取电机实时位置）
 ├── 包含 0x37（读取电机位置误差）
 ├── 包含 0x39（读取驱动温度）【仅X42S/Y42】
 ├── 包含 0x3A（读取电机状态标志）
 ├── 包含 0x3B（读取回零状态标志）
 └── 包含 0x3C（读取回零状态标志+电机状态标志）

其中 0x3C 又包含：
 ├── 0x3A（电机状态标志）
 └── 0x3B（回零状态标志）
```

#### 只能通过0x43读取的参数（无单独读取指令）

0x43返回的参数中，绝大多数都有对应的单独读取指令，仅以下参数**没有单独读取指令**，只能通过0x43批量读取获取：

| 参数 | 固件 | 说明 |
|------|------|------|
| 编码器原始值 | 仅X固件 | X固件0x43返回的第4个参数，Emm固件无此参数 |

> 其余所有被0x43包含的参数（总线电压、总线电流、相电流、线性编码器值、目标位置、实时转速、实时位置、位置误差、温度、回零状态、电机状态）均有单独读取指令。

#### 不被0x43覆盖的指令（需单独读取）

以下4条单项读取指令**不被0x43包含**，必须单独发送命令读取：

| 功能码 | 指令名称 | 固件 | 发送格式 | 说明 |
|--------|----------|------|----------|------|
| 0x32 | 读取输入脉冲数 | 通用 | `Addr 32 6B` | 主机通过脉冲接口输入的脉冲累计数，默认16细分下3200脉冲=1圈 |
| 0x34 | 读取电机实时设定目标位置 | 通用 | `Addr 34 6B` | 电机实时设定的目标位置，X/Emm角度计算不同 |
| 0x38 | 读取电池电压 | Y42 | `Addr 38 6B` | 电池电压，单位mV，仅Y42型号支持 |
| 0x3D | 读取引脚IO电平状态 | X42S/Y42 | `Addr 3D 6B` | 各引脚电平状态，包括En/Stp/Dir引脚 |

> 注意：0x34在X固件下被0x43覆盖（包含在X固件的12参数中），但在Emm固件下不被覆盖。因此Emm固件用户如需读取"电机实时设定目标位置"，需单独发送0x34。

### 6.2 限电流/限速变体的包含关系

```
基础命令 ──→ 增强命令

0xFB（直通限速位置模式） ──→ 0xCB（+最大电流限制）【仅X固件】
0xFD（梯形曲线位置模式） ──→ 0xCD（+最大电流限制）【仅X固件】
0xF5（力矩模式）         ──→ 0xC5（+最大速度限制）【仅X固件】
0xF6（速度模式）         ──→ 0xC6（+最大电流限制）【仅X固件】
```

#### 基础命令与增强命令的区别

基础命令和增强命令的区别在于**限制参数的配置方式**不同，分为两个层级：

**层级一：全局配置（持久化存储）**

通过驱动配置参数设置，掉电不丢失：

| 功能码 | 参数 | 说明 |
|--------|------|------|
| 0x45 | 闭环模式最大电流 | 全局电流上限，默认3000mA，范围0-5000mA |
| 0x42/0x48 | 闭环模式最大速度 | 全局速度上限，默认3000RPM |

**层级二：命令级限制（每次发送时指定）**

通过增强版命令在每次发送时单独指定：

| 基础命令 | 增强命令 | 命令内限制参数 | 范围 |
|----------|----------|---------------|------|
| 0xF6 速度模式 | 0xC6 速度模式限电流 | 最大电流 | 0-5000mA |
| 0xF5 力矩模式 | 0xC5 力矩模式限速 | 最大速度 | 0-3000.0RPM |
| 0xFD 梯形位置模式 | 0xCD 梯形位置模式限电流 | 最大电流 | 0-5000mA |
| 0xFB 直通限速位置模式 | 0xCB 直通限速位置模式限电流 | 最大电流 | 0-5000mA |

#### 层级关系

```
基础命令（0xF6/0xF5/0xFD/0xFB）
  └─ 电流/速度限制来自全局配置（0x45/0x42）
  └─ 适用于限制值固定的场景

增强命令（0xC6/0xC5/0xCD/0xCB）
  └─ 电流/速度限制来自命令内参数
  └─ 覆盖全局配置，每次可以不同
  └─ 适用于需要动态调整限制的场景
```

#### 使用示例

| 场景 | 推荐命令 | 说明 |
|------|----------|------|
| 固定电流运行 | 0xF6（基础速度模式） | 电流受全局0x45限制 |
| 每次运动不同电流 | 0xC6（增强速度模式） | 每次命令指定不同电流值 |
| 固定速度力矩控制 | 0xF5（基础力矩模式） | 速度受全局0x42限制 |
| 每次不同速度上限 | 0xC5（增强力矩模式） | 每次命令指定不同速度值 |
| 固定电流位置运动 | 0xFD（基础位置模式） | 电流受全局0x45限制 |
| 精细控制位置电流 | 0xCD（增强位置模式） | 每次命令指定不同电流值 |

#### 注意事项

- 两种方式可以混用，根据需要选择基础命令或增强命令
- 增强命令的限制值**仅对当次命令有效**，不影响全局配置
- 增强命令为X固件独有，Emm固件只有基础命令

## 附录：固件差异快速判断

### 判断当前固件类型

发送：`01 1F 6B`（读取固件版本）

返回中的bit1指示固件类型：
- bit1 = 0 → X固件
- bit1 = 1 → Emm固件

### 常用差异速查

| 场景 | X固件 | Emm固件 |
|------|-------|---------|
| 读取位置 | 0x36，返回÷10 | 0x36，返回×360/65536 |
| 读取转速 | 0x35，返回÷10 | 0x35，直接使用 |
| 速度控制 | 0xF6，速度×10 | 0xF6，速度直接用 |
| 位置控制 | 0xFD，角度×10 | 0xFD，脉冲直接用 |
| 快速位置 | 0xF1+0xFC，角度 | 0xF1+0xFC，脉冲 |
| PID | 位置Kp×2+速度Kp/Ki | Kp+Ki+Kd |

## 附录：多字节参数字节序说明

电机通讯协议中，所有多字节参数均采用**小端序（Little-Endian）**，即**低字节在前、高字节在后**。

例如总线电压字节序列 `5C 67`，实际值 = `0x675C` = 23655(mV)。

**适用场景**：
- 0x43 读取系统状态参数中的多字节返回值（电压、电流、位置、转速等）
- 0x42/0x48 驱动配置参数中的多字节参数（电流、速度、时间等）
- 运动控制命令中的位置角度（int32_t）、速度、加速度等参数

解析时需将接收到的字节序列按**低字节→高字节**顺序组合。
